# 🎬 Movie Recommendation System

A demo recommender system built with **FastAPI**, **PyTorch**, and **FAISS**.  
This project showcases how to build and deploy a scalable recommendation pipeline with a styled web interface.

---

## ✨ Features
- Two‑Tower neural network for user/item embeddings
- FAISS similarity search for fast recommendations
- FastAPI backend with Swagger UI (`/docs`)
- Colorful HTML homepage with search box
- Modular code structure (`src/`, `api/`, `models/`, `data/`)
